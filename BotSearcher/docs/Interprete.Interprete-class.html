cargarInstruccion(self, carpeta, modulo, clase)

 

Carga una instruccion al repertorio.

Parameters:
  • carpeta (string) - carpeta donde se encuentra el modulo.
  • modulo (string) - Fichero *.py que contiene la instruccion.
  • clase (string) - Nombre de la clase deribada de Instruccion.
Returns:
Devuelve None si hay algun problema. En otro caso devuelve una instancia de la funcion.

cargarRepertorio(self, xmlfile)

<instrucciones> <instruccion nombre='EjemploClase' modulo='EjemploModulo' path='./'/> </instrucciones>

Parameters:
  • xmlfile (string) - Nombre del fichero xml a cargar.

checkInstr(self, InstrClass)

 

Metodo que comprueba si una clase cargada es una instruccion valida para la ejecucion en el interprete.

Parameters:
  • InstrClass (Instruccion.) - clase a comprobar.
Returns:
Devuelve Cierto si la clase cargada es valida. Falso en otro caso.

ejecHelp(self, instr)

 

Ejecuta la instruccion help() de una Instruccion dada.

Parameters:
  • instr (string) - Nombre de la instruccion.

ejecInstr(self, instr, params)

 

Ejecuta una instruccion dado un nombre con unos parametros dados.

Parameters:
  • instr (string) - nombre de la instruccion a ejecutar
  • params (string) - Texto con los parametros de ejecucion.
Returns:
Devuelve Falso si la instruccion no se encuentra en el repertorio. Si se encuentra, devuelve el valor de la ejecucion de la instruccion.

ejecutarBucle(self)

 

Bucle principal de ejecucion del interprete. Lee las instrucciones y las ejecuta. Usando Exit salimos el programa.
