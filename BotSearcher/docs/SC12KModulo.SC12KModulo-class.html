getParamValue(self, key)

 

Este metodo no puede ser sobrescrito. Devuelve el valor de un parametro concreto, dada una clave.

Parameters:
  • key (string) - Clave del diccionario de _atributos. Debe estar previamente inicialiada en init().
Returns:
Si no existe la clave devuelve None, en otro caso devuelve el valor guardado del atributo.

init(self)

 

Este metodo debe ser sobrescrito. Es la inicializacion del modulo. Debe anadir como minimo la lista de parametros al diccionario self._atributos.

Es una Inicializacion inicial, se debe tener en cuenta que posteriormente se modificaran los atributos y se ejecutara inicializa antes de la primera vuelta de ejecucion.

reinicializa(self)

 

Este metodo no puede ser sobrescrito. Metodo que utiliza inicializa y lo marca como inicializado si es correcto.

Returns: boolean
  • Cierto si la inicializacion ha sido correcta.
  • Falso en otro caso.

Este metodo no puede ser sobrescrito. Establece el valor de una clave con el valor dado.

  • value (all) - Valor que se quiere dar al atributo con nombre <key>
Returns:
