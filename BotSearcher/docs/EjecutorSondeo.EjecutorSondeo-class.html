Class EjecutorSondeo

Modelo para modulos que implementen algun tipo de rastreo sobre 1 direccion IPv6. Estos modulos deben generar resultados compatibles con el almacen de informacion centralizada.

Cada implementacion debe reimplementar los metedos: init(), inicializa(), ejecutarPaso(), getResultInfo()
