incrementarIPv4(self)

 

Incrementa la direccion IPv4 numerica interna.

Cuando el digito de menor peso llega a 255 se pone a 0 y actualiza el digito posterior. De esta forma, cuando se llega al limite, la direccion dada sera 256.0.0.0 esto indicara que se ha recorrido todo el rango.
