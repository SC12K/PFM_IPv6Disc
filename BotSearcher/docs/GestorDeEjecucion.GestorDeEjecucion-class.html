Class GestorDeEjecucion

Gestor para controlar la ejecucion de las sondas a traves de una cola de ejecucion. Se pueden poner sondas en cola y se ejecutaran de forma secuencial. Por un lado almacenara las sondas a ajecutar y se propagara la informacion a todos los escritores.
